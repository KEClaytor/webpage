Matlab Component Runtime (MCR) Installer

Introduction and Download

The Matlab Component Runtime (MCR) is needed to run the .exe files on this page that have been generated by Matlab. You only need to install MCR once, and then all the programs will run.

However, my allocated server space is not large enough to post the ~500 Mb installer here. Instead I have provided several links to other pages where you can obtain the installer below.
